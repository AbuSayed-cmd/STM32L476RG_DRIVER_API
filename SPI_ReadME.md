## SPIx Peripheral
SPIx peripheral consist of followings-
* SPIx- Device Mode: SPIx Device is used to select mode as Master or Slave mode. When it is in master mode, it will produce serial clock otherwise it will not.
* SPI_Bus Config:
* SPI_DFF(Data Format):
* SPI_SCLK:
* SPI_CPOL:
* SPI_CHPA:
* SPI_SSM
